# To-Do List Application

A web-based To-Do List application developed using Python, Flask, MySQL, HTML, and CSS. The application helps users organize and manage their daily tasks through a simple and intuitive interface.

## Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* Mark tasks as completed
* Store task data in MySQL database
* Responsive and user-friendly interface

## Technologies Used

* Python
* Flask
* MySQL
* HTML5
* CSS3
